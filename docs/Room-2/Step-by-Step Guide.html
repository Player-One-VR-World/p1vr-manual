Step-by-Step Guide

First, launch the Another World application, called AW Device Manager, found on the Desktop.

Click the Launch button.

Select the game.
Shmooter
Kernel

info

Currently we only offer the games Schmooter and Kernel.

Setting Up Shmooter​

Select Shmooter on the launcher.

When launching the game you will be prompted with the following screen:

warning

Do not start the game with Default checked, this will cause the game to fail it's launch.

Begin to configure the pregame launch settings.

info

Configure the highlighted section in green to:
Lobby
Selected gamemode they are playing (Team Deathmatch, Deathmatch, etc.)
Default Lobby (prefer to choose themed lobby for special events)

info

Configure the highlighted section in yellow to:
DEFAULT: Welcome to Player One VR World!
If there is a special event, change to accomodate for that event.
EXAMPLE: Happy Birthday Shaq!

Match Configurations​

info

Each player can be configured separately:
Controller/Headset Battery Levels
Player Gamer Display Name
Team Color (only if playing Team Deatchmatch)

Make sure to calibrate each player headset before proceeding. Refer to the Calibrating Headsets page.

To launch the players into a game, press the cog next to the game in the Another World Application.
This will give you settings to once again change the map, gamemode, and configure game settings.

Setting Up Kernel​

note

Kernel plays on a smaller space than shmooter. Therefore, we only reccommend this game to a maximum of 4 players (2v2).

Select Kernel on the launcher.

When launching the game you will be prompted with the following screen:

warning

Do not start the game with Default checked, this will cause the game to fail it's launch.

Begin to configure the pregame launch settings.

info

Ensure that arcade mode is turned on as shown below. Arcade mode allows the players to just point down to reload their weapons.

If adult players prefer a more realistic version, consider keeping arcade mode off. Doing this means that you will have to remove the player's controllers from the guns, as their reloads will have to manually be done.
